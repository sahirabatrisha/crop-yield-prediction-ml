# Crop Yield Prediction Using Machine Learning

## Overview

This project applies machine learning techniques to predict crop yield per hectare using Malaysian agricultural and weather data.

The objective is to explore how environmental factors influence crop productivity and develop predictive models.

## Dataset

The project uses:
- Crop production data
- Cultivation area data
- Weather information

## Machine Learning Models

Models evaluated:

- Linear Regression
- Random Forest
- Gradient Boosting

## Workflow

1. Data preprocessing
2. Data integration
3. Feature engineering
4. Model training
5. Model evaluation

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Results

Random Forest:
- MSE: 0.001119
- R² Score: 0.266
